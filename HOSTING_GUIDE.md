# 🤖 Hosting Your Ayush Bot (ProChat AI)

Bhai, aapka chatbot host hone ke liye ekdum taiyar hai! Kyunki main suraksha (security) ki wajah se aapke computer se file 'select' karke upload nahi kar sakta, aapko bas ye 10-second ka kaam karna hai:

### **1. Netlify par Host karein (Free & Permanent)**
1.  Apne browser mein ye link kholein: **[https://app.netlify.com/drop](https://app.netlify.com/drop)**
2.  Apne desktop par **`bot`** folder ke andar jo **`ayush-bot.zip`** file maine banayi hai, usey pakad kar iss website ke upar **Drop** kar dein.
3.  Bas! Aapka chatbot LIVE ho jayega.

### **2. Unique Name kaise rakhein?**
-   Deploys hone ke baad, Netlify aapko ek link dega (jaise `something-unique.netlify.app`).
-   Wahan "Site Settings" -> "Change Site Name" par ja kar aap **`ayush-pro-ai`** ya koi bhi apna unique name rakh sakte hain.
